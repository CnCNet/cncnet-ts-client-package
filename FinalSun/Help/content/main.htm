FinalSun is a map and mission editor for Tiberian Sun. You can download the newest version of it here. This manual was written for FinalSun 1.0.

Make sure you at least read the tutorials, not doing this very often results in "internal errors". The main tutorial is located here. It shows you how to do a completely new multiplayer map with bridges, water and cliffs. Very detailed, many pictures and therefore really recommended!

You should also read the frequently asked questions which should help you.

You need to agree to the license if you want to use this editor. Else you are not allowed to use it. If you start the editor, you agree to the license.


Manual notes:

You should use at least Internet Explorer 4.x or Netscape 4.x to view this manual. I recommend turning on JavaScript for Netscape, as this enables better display of the manual.
